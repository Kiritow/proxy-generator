# Simple Nginx Config Generator

Generate nginx `http` section config from simple YAML configurations.

Support [acme.sh](https://github.com/acmesh-official/acme.sh) as the certificate management tool.

## Install

```bash
git clone https://github.com/Kiritow/proxy-generator
cd proxy-generator
./install.sh
```

## Example Config

See [config.yaml.example](config.yaml.example) for more information.

## Run Nginx Container

`start.sh` is provided as a minimal start script.
